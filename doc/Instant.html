Inlining C/C++ in Python

1. Summary

The Instant module is a small python module that allows C and C++ code to be inlined in Python to enhance performance. The C/C++ code and the wrapper code generated by SWIG is compiled and linked into a shared library loadable in Python. 
>>> import Instant, os  
>>> ext = Instant.Instant()
>>> c_code = """
int foo(){
  return 2222;
}
"""
>>> ext.create_extension(code=c_code, module='test1_ext')
>>> from test1_ext import foo 
>>> print foo() 
2222
